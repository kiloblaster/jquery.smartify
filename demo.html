Add, Remove or Toggle classes example

Add class
<div class="smartify" data-add-class="alert alert-success">
    …
</div>
Supurb! class `alert` and `alert-success` appended successfully.
Remove and Add class together
<div class="smartify alert alert-danger" data-remove-class="alert-danger" data-add-class="alert-info">
    …
</div>
Supurb! class `alert-danger` replaced by `alert-info`.
